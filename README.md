# RevEthCN Presale DApp

Presale and Swap interface for the RVCN token on BNB Smart Chain.

## Features

- 🔁 Swap RVCN ↔ BNB using PancakeSwap V2
- 💰 Presale Purchase Interface
- 📊 Live Dashboard with Token Balances
- 📱 Mobile-Responsive
- 🔒 MetaMask Integration
